ADD2MINE

An E-Commerce App, that sells trendsetting clothes.

Visit Site

DESCRIPTION

This is an E commerce App made with React Library. It was made when the developer was in the process of learning React. The app is like any other e commerce app. User can register, sign-in and add to cart any items that he/she chooses. It has also Stripe Payment Integration which allows the user to use credit card.
